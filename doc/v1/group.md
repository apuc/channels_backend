## Группы

### Получение групп пользователя

Для получения групп пользователя необходимо отправить **`GET`** запрос на **`/v1/group`**

Пример ответ:<br>
```
{
    "data":[
        {
            "group_id": 1,
            "title": "Ms.",
            "slug": "ms",
            "status": "active"
        },
        {
            "group_id": 2,
            "title": "Ms.",
            "slug": "ms",
            "status": "disable"
        }
    ]
}
```
